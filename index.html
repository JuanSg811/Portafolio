Proyectos

Aplicación en Java

Desarrollé una aplicación Java completa para búsqueda, traducción y gestión de letras musicales, integrando múltiples APIs externas con una base de datos local usando SQL. La plataforma incluye autenticación de usuarios, interfaz gráfica intuitiva y funcionalidades avanzadas de procesamiento de texto.

Optimización de Transporte

Desarrollé una solución integral para optimizar el sistema de transporte público del Área Metropolitana de Monterrey mediante la aplicación del algoritmo de Kruskal para calcular el Árbol de Expansión Mínima (MST). El proyecto incluyó tanto el análisis teórico del problema de movilidad urbana como la implementación práctica de un programa en C para resolver problemas de optimización de redes.

Análisis Numérico

Desarrollé una aplicación gráfica completa en Python para el análisis de datos de consumo eléctrico, implementando métodos numéricos avanzados con una interfaz de usuario intuitiva. La aplicación permite visualizar, analizar y procesar datos de consumo energético mediante técnicas computacionales.
